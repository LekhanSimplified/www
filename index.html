Lekho

Lekho is a website building software which is based on Pareto Principle. It states that 80% of output can be achived with 20% of inputs.

There are 100s of HTML tags and 200s of CSS commands, but only 10s of HTML tags and 30s of CSS Commands are used to build almost any website.

Our product lists these commonly used tags and commands along with some readymade components such as gallery, dropdown menu which can be used to build almost any website.
