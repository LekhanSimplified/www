Website

Lekho Website
makes it easy to design your homepage. It is based on the Pareto Principle, which states that 80% of output can be achieved with 20% of input.

A website can be built with just 10 HTML tags and 30 CSS commands, but there are hundreds of HTML tags and double CSS commands.

Besides these tags and commands, we've got some ready-made components like galleries and dropdown menus that can be used to build almost any website.
